# 🔲 2D Arrays

Matrix-based problems — covering creation, traversal, searching, and finding extremes in a 2D array.

---

## 📂 Problems

| # | Problem |
|---|---------|
| 1 | Creation of matrix |
| 2 | Search an element in matrix |
| 3 | Largest and smallest element in matrix |

---

## 🎯 Key Concepts / Learnings

- A 2D array (matrix) is essentially an array of arrays, accessed using row and column indices.
- Searching an unsorted matrix requires checking every element — O(rows × columns).
- Finding the largest/smallest element requires a single full traversal, tracking the current max/min while iterating.

## 🎯 Key Concepts / Learnings

- Arrays store elements in contiguous memory, allowing O(1) index-based access.
- Passing arrays to functions passes a reference, so changes inside the function reflect outside.
- Linear search checks each element one by one — O(n) time.
- Binary search works only on sorted arrays and repeatedly halves the search space — O(log n) time.
- Sub-arrays are contiguous portions of an array; generating all of them typically takes O(n²) or O(n³) depending on the approach.
